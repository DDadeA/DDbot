# DDbot
Discord tts bot written in python

## 1. Quick Start
```
git clone https://github.com/DDadeA/DDbot
cd DDbot
python -m ./DDBOT.py
```
## 2. Requirements
 - python>=3.7
 - ffmpeg

Below required modules will be automatically installed.
```
discord==2.2.3
PyNaCl==1.5.0
PySimpleGUI>=4.60.5
gTTS==2.3.2
navertts==0.3.1
audio-effects==0.22
pydub==0.25.1
GitPython
```
